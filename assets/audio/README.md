# 🔊 오디오 에셋 폴더 (assets/audio/)

이 폴더에 아래 **5개 사운드 파일**을 넣으면 게임에 자동으로 매칭되어 재생됩니다.
파일이 아직 없어도 게임은 정상 작동하며, 사운드만 조용히 비활성화됩니다.

| 파일명 | 용도 | 재생 방식 |
| :--- | :--- | :--- |
| `bgm_sketch_world.mp3` | 메인 배경음악 (퐁퐁 스케치 월드) | 무한 루프 |
| `sfx_jump.mp3` | 몽이 점프 | 1회 |
| `sfx_purify.mp3` | 발판 정화 (흑백 → 파스텔) | 1회 |
| `sfx_stomp.mp3` | 지우개 몬스터 밟기 | 1회 |
| `sfx_victory.mp3` | 100% 정화 클리어 팡파르 | 1회 |

> 파일명은 `lib/helpers/audio_manager.dart`의 상수와 정확히 일치해야 합니다.
> 위 표의 이름 그대로 이 폴더에 복사해 넣으면 바로 인식됩니다.
